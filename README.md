# Missed Image Viewer website

Static GitHub Pages landing page for Missed Image Viewer, a lightweight macOS image viewer.

## Pages

- `index.html` - marketing landing page with Mac App Store CTA.
- `contact.html` - support contact page using `mailto:kaverynkiryl@gmail.com`.
- `privacy.html` - privacy policy.
- `terms.html` - redirects to Apple's Standard EULA.

## Local preview

Open `index.html` directly in a browser, or run:

```sh
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## App Store link

The Mac App Store button points to `https://apps.apple.com/app/id6755685790`.
